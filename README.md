# 会计业务翻译器

一个静态前端会计业务处理辅助工具，用于把常见现实业务转换成原始凭证、会计分录、借贷逻辑和财务报表列示。

## 功能

- 业务转分录
- 原始凭证清单
- 财务报表列示
- 科目速查
- 处理记录本地保存
- 处理记录 TXT 导出
- 完整案例演示

## 技术栈

- HTML
- CSS
- JavaScript
- localStorage 本地保存

## 安全说明

当前版本为静态前端网站，数据默认保存在用户浏览器本地，不会主动上传到服务器。请勿输入真实客户隐私、银行账号、身份证号、完整税号或公司敏感凭证信息。

本工具用于学习、作品展示和基础业务处理思路辅助，不构成正式财务、税务或审计意见。

## 本地运行

直接打开 `index.html`，或使用本地服务器运行：

```bash
npx serve .
```

## 发布

可部署到 GitHub Pages、Netlify、Vercel 或 Cloudflare Pages。
